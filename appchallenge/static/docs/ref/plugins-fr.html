Plugiciels

Vous cherchez la BOEW v3.1?

À partir du 23 septembre 2014, la version 3.1 de la Boîte à outils de l'expérience Web n'est plus supportée. Le code source et la documentation ont été déplacés vers le dépôt wet-boew-legacy.

Needs translation A WET plugin provides various types of functionality useful to a website and provided by calling the function in the class attribute of an HTML tag.

  • Alertes repliables - Créer des alertes que les utilisateurs peuvent replier.
  • Basculer - Plugiciel qui permet un lien à basculer des éléments entre les états activé et désactivé.
  • Calendrier d'événements - Un calendrier servant à consulter une liste d'événements.
  • Chronologie intégrée Twitter - Affiche des chronologies intégrées Twitter.
  • Contenu par pays - Un enveloppeur de AjaxLoader qui insère de contenu télécharger via AJAX. Le contenu est basée sur l'emplacement des visiteurs comme déterminé par http://freegeoip.net
  • Contenu superposé - Un plugiciel fournissant un modèle de contenu superposé flexible et réactif.
  • Data Ajax - Un enveloppeur simple pour AjaxLoader qui insère de contenu télécharger via AJAX.
  • Data Inview - Affiche de contenu superposé quand une section se déplace hors de la clôture.
  • Data Picture - Conversion de Picturefill guidé par l'événement.
  • Égalisation des hauteurs - Un plugiel d'égalisation des hauteurs réactif.
  • Expiration de la session - Ferme la session de l'utilisateur après un certain délai.
  • Favoricône - Offre la possibilité d'ajouter et de mettre à jour les favoricônes d'une page.
  • Fils de syndication - Affiche les articles d’un ou plus fils de syndication.
  • Formulaire de rétroaction - Un formulaire normalisé pour soumettre une rétroaction au sujet d'une page ou d'un site Web particulier.
  • Gadget de partage - Permet à l'utilisateur de partager un contenu Web sur les plateformes de médias sociaux.
  • Géocarte - Affiche une carte dynamique sur laquelle des informations de sources supplémentaires peuvent être superposées.
  • Graphiques - Génère des graphiques à partir de données d’un tableau.
  • Interface à onglets - Comprime plusieurs sections de contenu dans le même espace physique et fourni des onglets qui permettent à l'utilisateur de choisi la section de contenu à afficher.
  • Lecteur multimédia - Un lecteur multimédia accessible permettant d'intégrer des contenus vidéo et audio sur les pages Web.
  • Lightbox - Crée une galerie photo sur une page Web.
  • Menu - Une barre de menu interactive comprenant des sous-menus optionnels.
  • Mise en surbrillance de texte - Met en surbrillance sur une page Web les mots choisis.
  • Notes de bas de page - Une méthode cohérente et facile d’accès pour répertorier les notes de bas de page.
  • Prettify - Affiche de la syntaxe des extraits de code source.
  • Tables - Integrates the DataTables plugin into WET providing searching, sorting, filtering, pagination and other advanced features for tables.
  • Validation de formulaires - Effectue la validation de formulaires Web selon un ensemble de règles de base avant qu'ils soient soumis.
